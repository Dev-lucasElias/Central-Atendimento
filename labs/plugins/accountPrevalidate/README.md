# Account PreValidate Lab

## Objetivo

Criar um plugin de estudo para validar dados antes da criacao de uma conta.

## Cenario

- Mensagem: `Create`
- Entidade: `account`
- Estagio: `PreValidation`

## O que estudar

- como acessar `Target`
- como validar campos obrigatorios
- quando lancar `InvalidPluginExecutionException`
- diferenca entre `PreValidation`, `PreOperation` e `PostOperation`

## Entrega esperada

- plugin simples em `src/`
- testes ou plano de teste em `tests/`
- anotacoes em `notes.md`
- aprendizados em `lessons-learned.md`
