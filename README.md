# Dynamics Study Repository

Repositorio para estudo pratico de Dynamics 365, Dataverse e customizacoes relacionadas.

## Objetivos

- Centralizar teoria, laboratorios e projetos aplicados.
- Registrar aprendizados, erros e decisoes tecnicas.
- Evoluir de conceitos basicos para cenarios proximos de projeto real.

## Estrutura

- `docs/`: conceitos, anotacoes e visao de arquitetura.
- `labs/`: experimentos pequenos e focados.
- `projects/`: cenarios mais completos.
- `snippets/`: exemplos reutilizaveis.
- `exercises/`: pratica guiada por dificuldade.
- `templates/`: bases para novos estudos.
- `assets/`: diagramas e imagens de apoio.

## Trilha sugerida

1. Estudar `docs/conceitos`.
2. Executar laboratorios em `labs/dataverse`, `labs/plugins` e `labs/javascript`.
3. Resolver exercicios em `exercises`.
4. Consolidar em projetos dentro de `projects`.

## Temas principais

- Dataverse
- Model-driven apps
- Plugins
- JavaScript Client API
- Power Automate
- Seguranca e roles
- Integracoes

## Convencoes

- Cada laboratorio deve ter `README.md`, `notes.md` e `lessons-learned.md`.
- Cada novo experimento deve responder: objetivo, setup, execucao e resultado.
- Snippets devem ser pequenos e com contexto de uso.
